# Odd Function
==2024-11-14

## Theory
If a function $f(x)$ is such that $f(-x)=-f(x)$, it is odd function.

---
## Examples
1. $\sin(x)$-> blue
2. $y=x$->purple
3. $y=x^3$->green

```desmos-graph
y=\sin(x)
y=x^3
y=x
```
