# Stack

```ad-question
title: What is stack?
Stack is a data-structure implemented for handling fast data retrieval for latest information.
```

```ad-note
title: Principle
Stack works on LIFO(Last in First Out).
The last element that went to stack comes out first.
```

![[Pasted image 20240801013414.png]]
```ad-note
title: Pro
- Fast data retrieval for the latest update.
```

```ad-note
title: Con
- Takes a lot of memory very fast and stack can grow very tall.
```

```ad-important
title: Stack Operations
### Push
Put Something in the stack.

---

### Pop
Remove the last in from the stack.
```


```ad-summary
- Using the stack is convenient, but there’s a cost: saving all that info can take up a lot of memory.
- A stack has two operations: push and pop.
- All function calls go onto the call stack.
- The call stack can get very large, which takes up a lot of memory.
```
