# Linked List

```ad-note
title: Defination
- Non contiguos cell connected with each cell containing item of that index and the address to the next cell.
```

```ad-important
- More memory used in comparision to arrays.
- Randomly inaccessible
- Can have any type
```

![[Pasted image 20240801000421.png]]

```ad-note
title: More Memory
- To store both values, and address of next value.
- Dynamic values.
```

```ad-note
title: Randomly inaccessible
The values should be read one after another- we don't know the address of value we want.
```

```ad-note
The values can be of any type.
As We are just linking blobs of memory, the values can have any memory size- they can be of any type.
```
