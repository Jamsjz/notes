# Function Checks
==2024-11-14

---
> Vertical And Horizontal Checks
---
## Vertical Line Check
```ad-info
title: Theory
To check if a curve is a function or not, you do a <u>vertical check</u>.

```ad-important
title: If a vertical line can intersect the graph at two points then it's not a function. Why?
Because if so, one input will give two output and That is not the defination of [[Function]]
```

```ad-example
title: $y=\sin x$ is a function
```desmos-graph
\sin x
x=4
```

```ad-example
title: $x^2+y^2=1$ is not a function
```desmos-graph
x^2+y^2=16
x=3
```
---
## Horizontal Line Check
```ad-info
title: Theory
To find if a function is one to one or not, use horizontal line check.
```ad-important
title: Why?
Because in graph, for different x, the y is same.
```
```ad-example
title: $y=x^2$ is a function but not one to one.
```desmos-graph
y=x^2
x=1
```
