# Continuity

==2024-11-17

---
## Theory
A function $f(x)$ is continuous at an interior point x = c if the following conditions holds.

1. $\lim_{ x \to c }f(x)$ exists (from both left and right side.)
2. $f(c)$ exists.
3. $\lim_{ x \to c }f(x)=f(c)$
