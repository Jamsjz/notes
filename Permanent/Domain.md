# Domain
==2024-11-13==

---
> $f\text{ : } X\to Y$  Domain (D): X
---
```ad-info
For a given [[Function]] $f(x)$, Domain is the set of all possible input for which f(x) is defined.
```

```ad-important
To Find Domain:
- Search conditions for which f(x) is not defined. 
```

```ad-example
$\text{Function: }f(x)=\frac{1}{x}$
```desmos-graph
y = \frac{1}{x}
```
